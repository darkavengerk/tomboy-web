* 노트
  * 타이틀 다음 줄은 폰트 및 줄크기 작게하기(2번째 줄은 필요할 때 쓰는 부제용 공간으로)
  * 데스크탑에서 툴 박스 높이 살짝 줄이기
  * 항상 위 기능(데스크탑)
  * 마우스 우클릭 시 메뉴 커스터마이즈(현재는 브라우저 네이티브 동작)
  * 데스크탑에서 상하좌우에서 노트 사이즈 조정 가능하도록 변경(현재는 우측 하단에서만 가능)
* 단축키 관련:
  * 노트 상단(제목표시줄)을 마우스 가운데 버튼으로 클릭해 제일 뒤로 보내기(데스크탑)
  * 날짜 넣기(ctrl+d 혹은 우클릭메뉴): 형식은 yyyy-mm-dd
  * 알트 + 좌우 화살표(혹은 우클릭메뉴)로 리스트 항목으로 만들거나 깊이 조정하기
* 개선
  * 리스트에서 탭 등으로 깊이 조정할 때 해당 항목만 변경 되게 하기(현재는 하위 항목들까지 같이 변경됨)
    * 하위 항목까지 같이 깊이가 조정되는 게 직관적으로는 맞는 것 같지만 실제 써보면 내가 선택한 부분만 조절되는 게 편함
  * 텍스트 복사하면 형식 제대로 맞춰서 클립보드에 넣기(우클릭 메뉴로 형식 선택 가능하게)
